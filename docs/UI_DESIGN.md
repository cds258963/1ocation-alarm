# 🎨 UI/UX 设计文档

## 设计风格

**设计语言：** Material Design 3 (Material You)

**设计理念：**
- 简洁直观，减少认知负担
- 地图为核心，突出位置信息
- 快速操作，3 步内完成闹钟创建

---

## 页面设计

### 1. 首页 (Home Screen) - 地图选点

**布局：**
```
┌─────────────────────────────┐
│  状态栏                      │
├─────────────────────────────┤
│  🔍 搜索地点...              │  ← 搜索栏 (悬浮)
├─────────────────────────────┤
│                             │
│                             │
│         🗺️ 地图区域           │
│                             │
│            📍               │  ← 中心标记
│                             │
│                             │
├─────────────────────────────┤
│  📍 当前位置：北京市朝阳区     │  ← 地址卡片
│     经度：116.4074           │
│     纬度：39.9042            │
├─────────────────────────────┤
│  [🎯 设置闹钟]  [📍 定位]     │  ← 底部操作栏
└─────────────────────────────┘
```

**交互说明：**
- 双击地图 → 选点并弹出地址卡片
- 拖拽地图 → 实时更新中心点坐标
- 点击"定位" → 回到当前位置
- 点击"设置闹钟" → 跳转闹钟设置页
- 点击搜索栏 → 打开搜索页

## 图标设计

**Material Icons 使用：**
- 地图：`map` / `place`
- 定位：`my_location` / `gps_fixed`
- 闹钟：`alarm` / `access_alarm`
- 设置：`settings`
- 搜索：`search`
- 删除：`delete`
- 编辑：`edit`
- 添加：`add` / `add_circle`
- 完成：`check` / `done`
- 通知：`notifications`

---

## 动效设计

### 页面切换
- 右滑返回：`SlideTransition`
- 底部弹出：`ModalBottomSheet`
- 淡入淡出：`FadeTransition`

### 按钮反馈
- 点击涟漪：`InkWell`
- 长按震动：Haptic Feedback

### 地图动画
- 定位平滑移动：`AnimationController`
- 缩放过渡：`AnimatedContainer`

---

## 响应式设计

### 横屏适配
- 地图占满左侧 2/3
- 列表/设置在右侧 1/3

### 平板适配
- 使用 `LayoutBuilder` 动态调整
- 双栏布局（列表 + 详情）
