# 📋 功能规格说明书 (Feature Specification)

## 产品定位

一款基于地理围栏的位置提醒应用，帮助用户在到达/离开指定地点时自动收到通知。

**目标用户：**
- 通勤族（公交/地铁下车提醒）
- 旅行者（景点打卡提醒）
- 商务人士（会议地点提醒）
- 老年人（活动范围安全提醒）
- 家长（孩子安全区域监控）

---

## 核心功能

### 1. 地图选点 ⭐⭐⭐⭐⭐

**功能描述：**
用户通过地图选择或搜索地点，设置闹钟位置。

**详细需求：**
- [ ] 支持高德地图展示
- [ ] 支持手势拖拽地图
- [ ] 支持双击地图选点
- [ ] 支持搜索 POI（兴趣点）
  - 公交站点
  - 地铁站
  - 商场/建筑
  - 自定义地址
- [ ] 支持当前位置定位
- [ ] 显示选点经纬度

**用户流程：**
```
打开首页 → 查看地图 → 搜索/选点 → 确认位置 → 进入设置页
```

---

### 2. 闹钟设置 ⭐⭐⭐⭐⭐

**功能描述：**
配置地理围栏参数和提醒方式。

**详细需求：**

#### 2.1 基础设置
- [ ] 闹钟名称（默认：地点名称）
- [ ] 提醒半径：250m / 500m / 1000m / 2000m / 3000m
- [ ] 触发条件：
  - [ ] 进入区域时提醒
  - [ ] 离开区域时提醒
  - [ ] 进入和离开都提醒

#### 2.2 时间设置
- [ ] 生效时间段（可选）
  - 例如：工作日 7:00-9:00（早高峰通勤）
- [ ] 生效日期（可选）
  - 每天 / 工作日 / 周末 / 自定义

#### 2.3 提醒设置
- [ ] 通知铃声（系统铃声/自定义）
- [ ] 振动模式
- [ ] 重复提醒次数（1-9 次）
- [ ] 提醒间隔（30 秒/1 分钟/5 分钟）
- [ ] 语音播报（可选）
  - "您已到达 XX 地点"

#### 2.4 高级设置
- [ ] 后台监控模式
  - 省电模式（5 分钟定位一次）
  - 平衡模式（1 分钟定位一次）
  - 高精度模式（30 秒定位一次）
- [ ] 低电量自动暂停（可选）

---

### 3. 闹钟列表 ⭐⭐⭐⭐

**功能描述：**
管理所有已创建的闹钟。

**详细需求：**
- [ ] 列表展示所有闹钟
  - 闹钟名称
  - 地点缩略图
  - 半径大小
  - 状态（启用/禁用）
- [ ] 滑动操作：
  - 左滑删除
  - 右滑编辑
- [ ] 批量操作：
  - 批量启用/禁用
  - 批量删除
- [ ] 搜索闹钟
- [ ] 按距离排序
- [ ] 按创建时间排序

---

### 4. 地理围栏监控 ⭐⭐⭐⭐⭐

**功能描述：**
后台持续监控位置，触发提醒。

**详细需求：**
- [ ] 后台位置监控
- [ ] 多围栏同时监控（最多 20 个）
- [ ] 进入/离开事件检测
- [ ] 触发通知推送
- [ ] 触发语音播报
- [ ] 触发记录日志

**技术要点：**
- Android 前台服务（保活）
- 电池优化白名单引导
- 各厂商后台管理适配（小米、华为、OPPO、vivo）

---

### 5. 通知提醒 ⭐⭐⭐⭐⭐

**功能描述：**
当地理围栏触发时，发送通知。

**详细需求：**
- [ ] 锁屏通知
- [ ] 状态栏通知
- [ ] 通知点击跳转详情页
- [ ] 通知样式：
  - 标题：到达/离开提醒
  - 内容：地点名称 + 时间
  - 图标：应用图标
- [ ] 语音播报（可选）
- [ ] 连续提醒（可配置次数）

---

### 6. 数据统计 ⭐⭐⭐

**功能描述：**
记录用户的闹钟触发历史。

**详细需求：**
- [ ] 触发记录列表
  - 时间
  - 地点
  - 类型（进入/离开）
- [ ] 统计图表
  - 本周触发次数
  - 常去地点 TOP5
- [ ] 足迹地图（可选）

---

### 7. 设置页 ⭐⭐⭐

**功能描述：**
应用全局设置。

**详细需求：**
- [ ] 通知设置
  - 允许通知开关
  - 通知铃声选择
  - 振动开关
- [ ] 定位设置
  - 定位精度选择
  - 后台定位开关
- [ ] 电池优化
  - 引导用户添加白名单
- [ ] 关于页面
  - 版本号
  - 开发者信息
  - 用户协议
  - 隐私政策

---

## 非功能性需求

### 性能要求
- 冷启动时间 < 2 秒
- 地图加载时间 < 3 秒
- 位置更新延迟 < 10 秒
- 后台耗电 < 5%/小时

### 兼容性要求
- Android 8.0+ (API 26+)
- 适配主流厂商 ROM（小米 MIUI、华为 EMUI、OPPO ColorOS、vivo Funtouch OS）

### 安全要求
- 用户位置数据本地存储
- 不上传用户隐私数据
- 符合 GDPR/个人信息保护法
